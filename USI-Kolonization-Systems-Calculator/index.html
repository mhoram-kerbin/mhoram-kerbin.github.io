USI Kolonization Systems Calculator

USI Kolonization Systems is an addon that provides an option to build 100% self-sufficient bases on other planetary bodies.
It packs a lot of modules that depend on each other. This site helps you with planning bases.

If you want to run for example a Modular factory at 150% capacity, enter '150' in the respective input field. The other rows will adjust and show you:

  • above: the requirements, that are necessary for this efficiency to be supported
  • below: the efficiencies this module can support.
